This website is based on the work made by Anton McConville and his analysis of David Bowie lyrics.

THE ORIGINAL PROJECT: LITTLE WONDER

The original project began as a personal way of remembering David Bowie, absorbing and reflecting his seasons and songs, but turned into an experiment to share.

It is most of all an examination of personas using the big five personality traits. David Bowie continually changed, gifting us many new characters and incredible songs, during a beloved career.

The objective of the project is to see if there is a correlation between his written lyrics and his visual personas.

We take the lyrics from albums that correspond to each persona/period and run Watson Personality Insights on them.

Clicking on one of the pictures of an artist on the main page opens another page that looks like a mock Twitter account for each persona. The 'tweets' contain sankey diagrams that break down the big five personality traits and a little bar chart that shows the combined trait measurements.

Each page has a video embedded, and has links to wikipedia articles about the albums from that persona. Listen to the music, follow the album links, read the personality data. How does the data correspond to that persona?

Displaying this as a Twitter page was a design idea to emphasize the theme of persona. This project considers changes too - it wants to reflect on changes in persona visually and through mining the data within the songs over time.

From the main view you can also find a link to a page which shows the big five traits over time, and per persona.

There are also controls to toggle between the contributing sub-traits. This view shows how we can track personality over time.

This project is built and runs happily on IBM Bluemix.
